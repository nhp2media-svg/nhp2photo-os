Decision Flow

  1. Is identity sacred? Enable Identity Lock. Avoid face replace unless mandatory.
  2. Is it historical? Use Archival mode. Keep changes minimal and physically plausible.
  3. Need angle/pose change? Start at 10–20°. Rebuild lighting physically for the new angle.
  4. Wax/plastic skin? Run Texture+Grain. Preserve pores and organic noise.
  5. Need resolution? Upscale last. Forbid invented micro-detail and sharpening halos.
