Decision Flow

Use this to pick the safest workflow for your goal.

  1. Is identity sacred? If yes, enable Identity Lock and avoid face replace unless mandatory.
  2. Is it historical/archival? If yes, choose Archival mode and keep changes minimal.
  3. Need angle/pose change? Start at 10–20° max. Rebuild lighting physically. Compare faces.
  4. Seeing wax/plastic skin? Switch to Texture+Grain task, preserve pores and noise.
  5. Need resolution? Upscale last. Forbid invented micro-detail and halos.
